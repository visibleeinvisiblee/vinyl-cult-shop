# VINYL CULT — сайт-вітрина

Мінімалістичний сайт з вкладками: **Лампи**, **Годинники**, **Стелаж**, **Наліпки**; лінійки **VinylCult** та **CyberCult**; плавні переходи; яскравість на фото лампи; анімація на годиннику.

---

## Що за що відповідає

| Елемент | Де змінювати | Опис |
|--------|----------------|------|
| **Меню (вкладки)** | Лампи, Годинники, Стелаж, Наліпки | Перемикання секцій з плавною анімацією. |
| **Лінійки** | Кнопки Усе / VinylCult / CyberCult | Фільтр товарів у вкладках Стелаж та Наліпки. |
| **Вкладка «Лампи»** | Фото: `images/lamp.jpg` | Ползунок **Яскравість лампи** змінює яскравість саме цього фото (filter: brightness). Додай своє фото лампи в папку `images/`. |
| **Вкладка «Годинники»** | Фото циферблата: `images/clock.jpg` (опційно) | Ползунок **Швидкість анімації** змінює швидкість обертання стрілки годинника. Фон циферблата — твоє фото або чорний. |
| **Стелаж / Наліпки** | У файлі `index.html` — блок `PRODUCTS_CONFIG` | Кожен товар має `category: 'shelf'` або `'stickers'` та `line: 'VinylCult'` або `'CyberCult'`. |
| **Товари (назва, опис, ціна, фото)** | `PRODUCTS_CONFIG` у `index.html` | Поля: `title`, `description`, `price`, `image`, `category`, `line`. |
| **Посилання «Замовити»** | Змінна `ORDER_LINK` | За замовчуванням — Instagram. |
| **Редагування без коду** | Кнопка «Редагувати контент» внизу справа | Зміна тексту/ціни товарів, збереження в localStorage. |

## Як змінювати товари через код

Відкрий `index.html` і знайди блок:

```javascript
const PRODUCTS_CONFIG = [
  {
    title: 'Назва',
    description: 'Опис.',
    price: '150 грн',
    image: 'images/product1.jpg',
    category: 'shelf',   // або 'stickers'
    line: 'VinylCult'    // або 'CyberCult'
  },
  // ...
];
```

- **title**, **description**, **price**, **image** — як раніше  
- **category** — `'shelf'` (зʼявляється у вкладці Стелаж) або `'stickers'` (Наліпки)  
- **line** — `'VinylCult'` або `'CyberCult'` (фільтр по лінійці)

Щоб змінити фото: заміни файли в папці `images/` (наприклад `product1.jpg`, `product2.jpg`) або зміни значення `image` у відповідному об'єкті.

## Як змінювати через сторінку (без коду)

1. Відкрий сайт у браузері.  
2. Натисни кнопку **«Редагувати контент»** внизу справа.  
3. Клікай по назві, опису або ціні товару і редагуй текст.  
4. Натисни **«Зберегти»** — зміни збережуться в цьому браузері.  
5. Натисни **«Вийти»**, щоб вимкнути режим редагування.

Фото так змінити не можна — тільки через заміну файлів у папці `images/` та оновлення шляху в `PRODUCTS_CONFIG` (або через майбутнє розширення).

## Як запустити локально

Просто відкрий файл `index.html` у браузері або запусти локальний сервер з папки проекту, наприклад:

```bash
# Python 3
python -m http.server 8000

# або npx (Node)
npx serve .
```

Потім відкрий у браузері: `http://localhost:8000`

## Як викласти на GitHub (новий репозиторій)

1. **Створи новий репозиторій на GitHub**  
   - Зайди на [github.com](https://github.com) → New repository.  
   - Назва, наприклад: `vinyl-cult-shop`.  
   - Не додавай README, .gitignore чи ліцензію — репозиторій має бути порожнім.

2. **У терміналі в папці проекту виконай:**

```bash
cd d:\COD\vinyl-heart\vinyl-cult-shop
git init
git add .
git commit -m "Vinyl Cult site: products, order button, sliders, Instagram"
git branch -M main
git remote add origin https://github.com/ТВІЙ_ЛОГІН/vinyl-cult-shop.git
git push -u origin main
```

Замість `ТВІЙ_ЛОГІН` підстав свій нік на GitHub. Якщо репозиторій створив з іншою назвою — підстав її замість `vinyl-cult-shop`.

3. **Увімкнути GitHub Pages**  
   - У репозиторії: Settings → Pages → Source: **Deploy from a branch** → гілка **main**, папка **/ (root)** → Save.  
   - Через кілька хвилин сайт буде доступний за посиланням типу:  
   `https://ТВІЙ_ЛОГІН.github.io/vinyl-cult-shop/`

## Структура проекту

```
vinyl-cult-shop/
├── index.html    # розмітка, стилі, скрипти, конфіг товарів і категорій
├── images/       # lamp.jpg (лампа), clock.jpg (циферблат), фото товарів
```
